# WebArena - Modos Claro/Oscuro

## Descripción
WebArena es una aplicación web interactiva que permite explorar diferentes paletas de colores populares para modos claro y oscuro. La aplicación proporciona una interfaz intuitiva para seleccionar temas de color, cambiar entre modos claro y oscuro, y personalizar colores individuales de la paleta.

## Características
- Selección de múltiples temas de color populares (GitHub, VS Code, Dracula, Solarized, One, Monokai, Material y Nord)
- Modo claro y oscuro para cada tema
- Previsualización móvil de los componentes con los colores del tema
- Selector de colores para personalizar la paleta
- Visualización de los colores de la paleta con sus valores hexadecimales
- Interfaz completamente responsive

## Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)

## Instalación
1. Clona o descarga los archivos del proyecto
2. Abre el archivo `index.html` en tu navegador web preferido
3. ¡Comienza a explorar los diferentes temas de color!

## Uso
1. Selecciona un tema de la lista disponible (GitHub, VS Code, Dracula, Solarized, One, Monokai, Material o Nord)
2. Elige entre el modo claro u oscuro
3. Visualiza la paleta de colores del tema seleccionado
4. Haz clic en cualquier color para personalizarlo con otro del tema o un color personalizado
5. Utiliza la previsualización móvil para ver cómo se verían los componentes con los colores seleccionados
6. Personaliza los colores y haz clic en "Actualizar Tema con Colores Seleccionados" para aplicarlos a la interfaz
